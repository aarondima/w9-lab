## Packer
Run the following commands in the packer directory
1. packer init .
2. packer validate .
3. packer build .

## Terraform
Run the following commands in the terraform directory
1. terraform init
2. terraform fmt
3. terraform validate
4. terraform plan -out plan.tf
5. terraform apply plan.tf
